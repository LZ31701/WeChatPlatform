MySQL显示命令
二、显示命令 
1、显示数据库列表。 
show databases; 
2、显示库中的数据表： 
use mysql;
show tables; 
3、显示数据表的结构： 
describe 表名; 
4、建库： 
create database 库名; 
5、建表： 
use 库名； 
create table 表名 (字段设定列表)； 
6、删库和删表: 
drop database 库名; 
drop table 表名； 
7、将表中记录清空： 
delete from 表名; 
8、显示表中的记录： 
select * from 表名
